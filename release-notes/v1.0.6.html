Focusflow 1.0.6

What's new:
- Added a new transcript toolbar with a microphone source dropdown and one-click transcript copy.
- Improved transcript summary reliability after meetings by preventing the inter-app summary listener from silently stopping.
- Added a richer Ask AI experience with a redesigned panel, typewriter-style response reveal, and improved feedback/copy interactions.
- Updated Note Details screens and supporting snapshot coverage for transcript, recording controls, and summary states.
- Refined event rows and timers, including clearer imminent/started states and better dashboard banner behavior.
- Suppressed meeting detection notifications when an event is about to start (within 15 minutes) to reduce noise.
- Introduced a YAML-configurable transcription policy for recording/silence/session behavior.
- Improved note details and popover UI behavior (including dynamic popover resize fixes and Today/Yesterday date headers).
- Improved resilience around offline auth/token refresh and coroutine error handling.
- Added multiple stability and visual polish fixes across Notes, Ask AI, and recording-related flows.
